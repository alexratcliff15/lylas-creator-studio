# Lyla's Creator Studio

UGC Creator Platform for Performance-Driven Content. Connects brands with creators, tracks Meta ad performance, manages commissions via Stripe Connect, and provides real-time dashboards for both creators and brand admins.

## Quick Start

```bash
npm install
cp .env.example .env   # then fill in your keys
npx prisma db push
npm run dev
```

See [DEPLOY.md](./DEPLOY.md) for full deployment instructions.
